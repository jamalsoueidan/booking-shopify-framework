# Interface naming rules

### Request define the query params for the api requirements, can be used in both frontend and backend.

```ts interface NameRequest {}

```

### Response define the query response for the api, can be used in both frontend and backend

```ts interface NameResponse {}

```

### Return define the internal express methods

```ts interface NameReturn {}

```
