<p>
  <b>Interface naming rules:</b>
</p>

<div>
  <i>
    Request define the query params for the api requirements, can be used in
    both frontend and backend.
  </i>
</div>
`interface NameRequest {}`

<br />
<br />

<div>
  <i>
    Response define the query response for the api, can be used in both frontend
    and backend
  </i>
</div>
`interface NameResponse {}`

<br />
<br />

<div>
  <i>Return define the internal express methods</i>
</div>
`interface NameReturn {}`
