---
description: Env that allows the use of CSS modules.
---

To tests components that use CSS modules, we will need to add the pattern for the CSS modules to the `transformIgnorePatterns` array in the Jest configuration.

This pattern will select all the CSS modules files with the following extensions:

- `.module.css`
- `.module.scss`
- `.module.sass`

```javascript
module.exports = {
  // ...
  "^.+\\.module\\.{css,scss,sass}$",
};
```

> Your IDE may complain about the import of the CSS modules. You can fix it by configuring Global declaration types in the root of your workspace.
